The Help page currently has 7 placeholder topic cards. The following specific documentation needs have been identified:

**Priority Score Calculation** — Explain the composite risk scoring used on the Compliance page's Priority Action Table. Document each factor and its point value (failing retro=100, missing=90, condition 1=80, damaged=70, condition 2=60, faded=50, overdue replacement=40, due within 90 days=30, never inspected=20, inspection >2yr old=10). Explain that points stack.

**Why:** Supervisors and directors will ask "why is this sign ranked #1?" The scoring needs to be transparent and defensible — especially if it's ever referenced in a liability case.

**How to apply:** When building out the Help page content, include a dedicated section on compliance scoring with examples.
